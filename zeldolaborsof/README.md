## Installation
Download the game from https://github.com/ZeldoRetro/labors_zeldo/releases and put in the `ports/zeldolaborsof` folder.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Interact|
|B|Attack|
|X|Item 1|
|Y|Item 2|
|START|Menus|
