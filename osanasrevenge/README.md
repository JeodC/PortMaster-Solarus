## Installation
Download the game from https://metaquarius.itch.io/osanas-revenge and add the .solarus file to `ports/osanasrevenge`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Use / Talk / Examine|
|B|Run|
|X / Y|Shoot|
|START|Pause Menu|
|SELECT|Save Menu|
